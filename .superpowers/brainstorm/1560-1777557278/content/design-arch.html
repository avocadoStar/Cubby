最终设计方案 — 第1节：总体架构

Cubby — Edge Favorites Clone 书签管理器

技术栈

前端
React 18 + TypeScript
Vite
Tailwind CSS (Fluent风格)
dnd-kit (拖拽)
react-virtuoso (虚拟列表)
Zustand (状态管理)
react-router (路由)
后端
Go + Gin
SQLite (WAL模式)
JWT (单用户密码认证)
LexoRank (排序)

系统架构

三层架构
Frontend Layer — React SPA, 单页应用
↓ REST API (JSON) + JWT Header
API Layer — Go + Gin, RESTful
↓ SQL
Data Layer — SQLite, parentId + sortKey
