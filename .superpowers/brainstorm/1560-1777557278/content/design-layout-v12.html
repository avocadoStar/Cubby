批量操作栏规范

触发条件 任意书签复选框被勾选（≥1项）
位置 右上角 absolute 定位 · right: 63px · top: 14px
角色 role="dialog" · aria-modal="true" · aria-label="alert dialog"
内容 "已选择 N 项" · [删除] primary · [取消] default
删除按钮 appearance="primary" · 蓝底白字 · 28px 高
取消按钮 白底灰边框 · 28px 高
消失 点击取消 / 执行删除后 / 取消所有选中

复选框三态完整流程

无选中 → 无 Alert Dialog，右上角仅显示工具栏默认状态
选中 1+ 项 → Alert Dialog 弹出，显示"已选择 N 项"，[删除] [取消]
点击[取消] → 所有复选框清空，Alert Dialog 消失
点击[删除] → 确认删除选中书签，Alert Dialog 消失
